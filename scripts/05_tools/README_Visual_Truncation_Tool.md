# Visual_Truncation_Tool.m

## Zweck
Interaktives Tool zum visuellen Trimmen von Impulsantworten.

## Eingaben
- `dataraw/*.mat` oder `processed/Proc_*.mat`

## Ausgaben
- Geänderte IR im Tool
- Optionaler Export aus dem Tool

## Nutzung
```matlab
run('scripts/05_tools/Visual_Truncation_Tool.m')
```

## Abhängigkeiten
- `functions/` im MATLAB-Pfad

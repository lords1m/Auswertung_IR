# example_ir_processing_pipeline.m

## Zweck
Zeigt typische Aufrufe der zentralen IR-Pipeline mit unterschiedlichen Optionen.

## Eingaben
- Beispiel-IRs im Skript oder per Datei

## Ausgaben
- Plots und Konsolenausgabe für die einzelnen Pipeline-Schritte

## Nutzung
```matlab
run('scripts/99_examples/example_ir_processing_pipeline.m')
```

## Abhängigkeiten
- `functions/` im MATLAB-Pfad
- `process_ir_pipeline`
